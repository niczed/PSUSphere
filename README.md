# PSUSphere

## Description
PSUSphere is a Django-based system for managing student organizations, colleges, programs, students, and organization memberships.

## Features
- Manage Colleges
- Manage Programs (linked to Colleges)
- Manage Organizations (linked to Colleges)
- Manage Students (linked to Programs)
- Manage Organization Memberships (Student ↔ Organization)
- Admin dashboard with search and filters
- Fake data generation using Faker (management command)
